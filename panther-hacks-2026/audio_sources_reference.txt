GODOT AUDIO BUS SETUP

Create these 4 buses in the Audio panel (bottom of Godot editor):

  Master
	Effects : Limiter (prevents clipping)
	Volume  : 0 dB

  Music     → sends to Master
	Effects : Reverb (room size 0.3, wet 0.2)
	Volume  : -5 dB

  SFX       → sends to Master
	Effects : Compressor (ratio 3:1, threshold -12 dB)
	Volume  : -3 dB

  Ambient   → sends to Master
	Effects : Reverb (room size 0.8, wet 0.5) — simulates underwater space
	Volume  : -10 dB

Assign each AudioStreamPlayer's "Bus" property in the Inspector
to match the bus names above.


RECOMMENDED FILE FORMATS FOR GODOT

  Music / long loops : .ogg  (smaller file, great for streaming)
  Short SFX          : .wav  (lowest latency, best for one-shots)

Enable "Loop" in the import settings for all looping files.
Enable "Stream" for music files to avoid loading them fully into RAM.
